Force Arcadium is now live

Ayan - 22nd September 2021

Public Release: What's New

The Force Arcadium, as stated, is a parent server to Help Force and all of the divisions which run similarly. Since the formation of the server, we've always aimed to establish a community which strives on building friendships through gaming. On the same lines of organizing things, we've taken a step further to introduce weekly game schedules. Each weekend, we'll be announcing a poll for you to choose that week's fun events and activities!

So the question being: Where does this leave the Help Force event managers?
Lucky for you, we've already thought about it! If you are an event manager interested in hosting fun events, you will be able to apply to host events in the Arcadium very soon. The application form releases this weekend, so keep an eye out!

Updates are gonna be the new constant for the Arcadium. The bulletin will consist of new games, events, and ideas each week. Suggestions for new games & events are always appreciated.

Our Vision

In an era where the virtual world is gaining importance, we want to stand out as a welcoming community for everyone. The Force Arcadium is not just a discord server, but a place for everyone to connect with each other through gaming. We plan to expand our community through popular games and create personalized content.

If you have any queries regarding the Arcadium, please do not hesitate to approach any staff member. Cheers to a spirited gaming community.
